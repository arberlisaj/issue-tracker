# Issue tracker - Full stack issue tracking application with user authentication.

[![Open in StackBlitz](https://developer.stackblitz.com/img/open_in_stackblitz.svg)](https://stackblitz.com/~/github.com/arberLisaj/issue-tracker)

## 🪄 Features

- ✅ CRUD operations.
- ✅ Built with Next.js.
- ✅ User authentication.
- ✅ Back-end using Prisma.

## 🧞 Commands

All commands are run at the root of the project from a terminal:

```sh
# install dependencies
npm install
```

```sh
# Start local development server at http://localhost:5173
npm run dev
```

```sh
# Build your production site to ./dist
npm run build
```

```sh
# Preview your build locallly before deploying
npm run start
```
